# PokeDream Data

Dados e progresso do bot idle do jogo [PokeDream](https://pokedream.com.br/).

Este repositório contém:
- **ROADMAP.md** — Estratégia de progressão e regras de decisão
- **GUIA.md** — Guia completo do jogo (raridades, tipos, hunts, linked tasks)
- **SOUL.md** — Configuração e instruções do Hermes Agent
- **PROGRESS.md** — Status atual do jogo (atualizado periodicamente)

> 📝 STATE.json e ACTIONS.json são voláteis (atualizados a cada 30-60s) e não entram no repositório.
